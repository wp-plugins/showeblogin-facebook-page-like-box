Brings the power of simplicity to display or embed Facebook Page Plugin (Like Box) widgets into your WordPress website by using latest Graph API v2.3.

== Description ==
As we know that the Facebook Like Box plugin is deprecated and therefore you all have to manually upgrade your FB like box to the Page Plugin. If not, your facebook Like Box plugin implementation will automatically fall back to the Page Plugin with effect from June 23rd 2015.

The **Showeblogin Facebook Page Plugin** is a special version of the Like box designed only for *Facebook Pages*. It allows WordPress admins to promote their Facebook Pages and embed a simple feed of content from a Page into websites by inserting widget in the sidebar or footer.

This plugin works with the responsive, fluid and static layouts and lets you easily embed and promote any Facebook Page besides yours own on your WordPress website. With the help of this plugin, your live visitors can like, comment and share all your Facebook posts in addition to liking and sharing your Facebook Page to their friends without having to leave your site.

= Features =
Following are the unique features of this plugin:-

* Show FB Like box for any Facebook Page.
* Change the width and height in accordance with your website.
* Show/Hide Facebook page Header Cover Photo.
* Show/Hide Facebook profile photos when friends like your FB page.
* Show/Hide latest posts from the Page's timeline.

== Installation ==
You may Install **Showeblogin Facebook Page Plugin** either via the WordPress.org plugin directory, or by uploading the files to your server.

= From your WordPress site Dashboard =

**Step-1:** Go to the "Plugins" menu in your WordPress site left sidebar and click "Add New" link.

**Step-2:** Search for "Showeblogin Facebook Page Plugin".

**Step-3:** Click on 'Install Now' button.

**Step-4:** Now, click on 'Activate' link.

**Step-5:** Finally, from widget area of your theme add Facebook Page Plugin and Enjoy!


= By uploading files to Server =

**Step-1:** Download the latest version of Showeblogin Facebook Page Plugin zip file.

**Step-2:** Unzip or Extract the plugin files.

**Step-3:** Upload the entire folder named showeblogin-facebook-page-plugin into your server under plugin folder (/wp-content/plugins).

**Step-4:** Now, login to your WordPress site and activate the Showeblogin Facebook Page Plugin and Enjoy!

Learn more (and get detailed instructions) in our [Tutorial guide](http://www.superwebtricks.com/blogger-beginner-guide/facebook-page-wordpress-plugin/ "Step by Step guide on how to install Showeblogin Facebook Page Plugin widget in WordPress").
